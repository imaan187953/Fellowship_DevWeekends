#  Deeper JavaScript

This repository contains my notes, code examples, and practice exercises for advanced JavaScript concepts. The goal is to understand how JavaScript works under the hood while learning modern ES6+ features and common programming patterns.

##  Topics Covered

- Prototype Chain
- ES6+ Features
  - Destructuring
  - Spread Operator
  - Rest Parameters
  - Optional Chaining
- Iterators
- Generators
- ES Modules
- CommonJS
- JavaScript Design Patterns


##  Learning Outcomes

By completing this repository, I will be able to:

- Explain the JavaScript prototype chain.
- Use modern ES6+ syntax confidently.
- Understand iterators and generators.
- Differentiate between ES Modules and CommonJS.
- Apply common JavaScript design patterns.
