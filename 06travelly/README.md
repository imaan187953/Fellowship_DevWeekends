# 🌍 Travelly

A modern and responsive travel landing page built with **React** and **CSS**. Travelly helps users explore beautiful destinations and provides an elegant interface for discovering travel experiences.

![Travelly Screenshot](./src/assets/travellweb.png)

---

## ✨ Features

- 🎨 Modern and clean UI design
- 📱 Fully responsive layout
- 🗺️ Travel destination showcase
- ⚡ Built with React functional components
- 🎯 Smooth and user-friendly navigation
- 🔥 Reusable and organized component structure

---

## 🛠️ Technologies Used

- **React.js**
- **CSS3**
- **JavaScript (ES6+)**
- **Vite** (if used as the build tool)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/travelly.git
```

### 2. Navigate to the project directory

```bash
cd travelly
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will run on:

```bash
http://localhost:5173
```
